# Music Player (HTML & CSS)

##  Project Description

This is a **simple music player** built using **HTML and CSS**.
It displays a clean music player card with a song title, cover image placeholder, and audio controls.

The project is beginner-friendly and helps understand:

* HTML structure
* CSS styling
* Using the `<audio>` tag

---

##  Technologies Used

* HTML5
* CSS3

---

##  Project Structure

```
music-player/
│── index.html
│── style.css
│── song.mp3
│── README.md
```

---

##  Features

* Gradient background
* Centered music player card
* Rounded album cover placeholder
* Built-in audio controls
* Responsive and clean UI
